Web pages

Your programming project next year (coursework worth 20% of the A level) will use:

  1. HTML (hypertext markup language) to define the content of web pages
  2. CSS (cascading style sheets) to define the content of web pages
  3. JavaScript to program the behaviour of web pages and generate dynamic content from a database

Javascript and Java are completely different languages
Use chrome to view your web pages. Hit F12 to open the developer tab.

  • The elements panel allows you to see the code for the web page.
  • The Console panel will be very useful when you write JavaScript. More on this next lesson.
  • The Network panel records all network requests. A single web page may have many requests (most usually GET and POST). Go to http://www.amazon.co.uk with this panel open and look at all
  • The Application page shows you cookies associated with the page.

You will find the Developer Tools in Chrome very useful!
